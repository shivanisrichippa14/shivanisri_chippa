### **Local Setup**
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/shivanisrichippa14/shivanisri_chippa.git
    cd shivanisri_chippa
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run development server:**
    ```bash
    npm run dev
    ```
4.  **Build production version:**
    ```bash
    npm run build
    ```
